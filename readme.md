=== wordpress-themedev-with-gulp ===

# Getting Started

### Required Components

- Node.js
    - <http://nodejs.org/>
    - v4.4.x is recommended.

## Set Up

#### 1) Install gulp

*Note:* If you have already installed gulp, skip this section.

    $ npm install -g gulp

#### 2) Clone

    $ git clone git@github.com:tsahala/wordpress-themedev-with-gulp.git {wp-theme-name}
    $ cd {wp-theme-name}
    $ npm install

#### 3) Run gulp

    $ gulp
